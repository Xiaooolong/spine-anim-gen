# Spine Anim Gen

A project to generate Spine animations from video inputs.

## Tech Stack
- React + TypeScript + Vite
- Express (Backend)
- PixiJS (Rendering)
- MediaPipe (Pose Extraction)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```
