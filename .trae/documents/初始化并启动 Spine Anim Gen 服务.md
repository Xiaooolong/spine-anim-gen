## 项目概述
本项目名为 **Spine Anim Gen**，是一个基于视频输入自动生成 Spine 骨骼动画的工具。

### 核心技术栈：
- **前端**：React + TypeScript + Vite + Tailwind CSS
- **后端**：Express + Node.js
- **图形处理**：PixiJS (用于预览和渲染)
- **动画提取**：MediaPipe (用于视频中的姿态提取)

### 主要功能：
1. **视频上传**：用户可以上传视频文件。
2. **姿态提取**：系统使用 MediaPipe 从视频中提取 2D 骨骼关键点。
3. **动画重定向**：将提取的姿态数据转换为 Spine 动画格式（JSON）。
4. **实时预览**：在浏览器中实时预览生成的 Spine 动画。

## 实施计划

### 1. 初始化项目
- 执行 `npm install` 安装所有必要的依赖包（包括前端和后端）。

### 2. 启动服务
- 执行 `npm run dev`。该命令将使用 `concurrently` 同时启动：
  - **前端开发服务器**：默认在 5173 端口。
  - **后端 API 服务器**：在 3003 端口（通过 `nodemon` 运行 `api/server.ts`）。

### 3. 验证与预览
- 确认后端 API（`/api/health`）和前端界面是否正常响应。
- 使用 `OpenPreview` 开启预览窗口，方便直接查看运行效果。
